## ROLE STRUCTURE CORRECTION

The system currently allows drivers to self-register.

This must be changed.

NEW WORKFLOW:

Admin creates drivers.

Driver cannot create an account themselves.

Admin workflow:

Admin Dashboard
→ Create Driver
→ Enter Driver Information
→ Upload Driver Documents
→ Create Login Credentials
→ Activate Driver

Driver receives:

* Username
* Phone
* Temporary Password

Driver logs in using credentials provided by Admin.

Driver changes password on first login.

Only Admin can:

* Create Driver
* Suspend Driver
* Activate Driver
* Verify Driver

This is required because transportation businesses must verify drivers before they are allowed to operate.
